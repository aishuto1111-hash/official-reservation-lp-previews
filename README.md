# 公式予約導線LP 公開用サイト

GitHub Pagesなどの無料静的ホスティングへ公開するためのディレクトリ。

## 構成

- `/index.html`: サービス説明ページ
- `/previews/c-nail-salon/`: C.向け個別プレビュー
- `/samples/brow-nail/`: 汎用サンプル

## 公開時の注意

- 店舗別プレビューは非公式ラフ案として扱う
- 店舗写真は使わず、仮写真のみ使用
- 公開情報をもとにした内容であり、正式導入時は店舗確認が必要
- 検索避けとして `noindex,nofollow` を入れている
- 送信前に公開URLを確認する

## 想定URL

GitHub Pagesで公開する場合:

- サービスページ: `https://<github-user>.github.io/<repo>/`
- C.プレビュー: `https://<github-user>.github.io/<repo>/previews/c-nail-salon/`
- 汎用サンプル: `https://<github-user>.github.io/<repo>/samples/brow-nail/`
